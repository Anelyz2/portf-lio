Mona Lisa Interativa: A Beleza do Código

O Projeto Mona Lisa no p5.js é uma recriação artística do famoso quadro de Leonardo da Vinci, feito com código. Em vez de usar uma imagem tradicional, o projeto utiliza formas geométricas (como círculos, retângulos, linhas ou pontos) para reconstruir a imagem da Mona Lisa de forma estilizada, pixelada ou pontilhista, explorando o poder do desenho generativo.

Objetivos principais:

Demonstrar manipulação gráfica com a biblioteca p5.js.

Explorar conceitos de computação visual, como leitura de pixels, cor e abstração.

Estimular a criatividade com código.

Procurando o Alvo: O Jogo Quente-Frio

O Projeto Quente e Frio no p5.js é uma experiência interativa onde o jogador tenta encontrar um alvo escondido na tela. A única forma de feedback é visual e sutil: uma bolinha é desenhada no cursor e seu tamanho aumenta conforme o mouse se aproxima do alvo.

Como funciona:

Um alvo secreto (ponto X/Y) é gerado aleatoriamente.

A cada quadro, o programa calcula a distância do mouse até o alvo.

Quanto menor a distância, maior o diâmetro da bolinha que aparece no mouse.

Isso simula a ideia de "quente" (bola grande, perto do alvo) e "frio" (bola pequena, longe do alvo), sem usar texto ou som.

Rápido e Reto: A Corrida Pura

Este projeto de jogo de corrida no p5.js simula uma corrida com vários objetos (carros, elementos de cenário, faixas da pista, etc.) se movendo na tela. O objetivo principal é criar uma sensação de velocidade e movimento contínuo, podendo ou não haver interação direta com obstáculos.

Como funciona:

Cenário em movimento: Elementos como linhas da pista ou paisagem se movem de cima para baixo, dando a ilusão de que o carro está se movendo para frente.

Vários objetos: Podem ser carros adversários, decorações, itens colecionáveis ou simplesmente elementos visuais.

Controle simples (opcional): O jogador pode mover seu carro para os lados usando setas, mas não é obrigatório evitar obstáculos.

Loop contínuo (não necessariamente): Objetos que saem da tela voltam para o topo, criando um ciclo infinito de corrida.
